Summary: Latitude vs X

The purpose of this project was to compare a variety of different weather variables against latitude as it approached the equater. In order to complete this analysis, I was ableto pull the weather data from the Open Weather API for over 500 randomly selected cities.


Using the Open Weather API, I gathered a variety of different weather variables inlcuding wind speed, cloudiness, temperature, and humidity. Then using MatPlotLib I was able to plot the data and make comparisions between latitude and each one of the weather variables. On this site you will be able to find a variety of different visualizations that were created as well as the source data with explanations and describtions of the correlations and trends that were uncovered.
